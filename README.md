# 복약 체크 (Pill Tracker)

매일 먹는 약을 기록하고, 남은 수량을 관리하는 아이폰용 PWA(웹앱)입니다.
모든 데이터는 기기(브라우저) 안에만 저장되며 서버·로그인이 필요 없습니다.

## 기능

- **오늘 탭** — 시간대별(아침/점심/저녁/취침 전) 복용 체크리스트. 체크하면 재고가 자동 차감되고, 체크 해제하면 복구됩니다.
- **내 약 탭** — 약 등록/수정/삭제, 재고 추가. 설정한 일수 이하로 남으면 경고 표시.
- **기록 탭** — 달력에서 날짜별 복용 현황 확인 (모두 복용 🟢 / 일부 🟡 / 안 먹음 🔴).

## 파일 구성

| 파일 | 역할 |
|---|---|
| `index.html` | 앱 뼈대 |
| `style.css` | 디자인 (모바일 우선) |
| `app.js` | 앱 로직 전체 (데이터는 localStorage에 저장) |
| `manifest.webmanifest` | 홈 화면 설치용 PWA 설정 |
| `sw.js` | 오프라인 지원 서비스 워커 |
| `icons/` | 앱 아이콘 |

## PC에서 미리 보기

```powershell
cd pill-tracker
npx serve .
```

또는 `index.html`을 브라우저로 바로 열어도 됩니다(서비스 워커만 비활성).

## 아이폰에 설치하기

1. 이 폴더를 **GitHub Pages** 또는 **Vercel**에 무료 배포 (HTTPS 주소 필요)
2. 아이폰 **Safari**로 해당 주소 접속
3. 공유 버튼 → **"홈 화면에 추가"**
4. 홈 화면 아이콘으로 실행하면 일반 앱처럼 전체 화면으로 동작

> ⚠️ 데이터는 기기에 저장되므로, Safari 설정에서 "방문 기록 및 웹 사이트 데이터 지우기"를 하면 기록이 사라집니다. (홈 화면 앱의 데이터는 별도 보관되어 비교적 안전합니다.)

## 알림 설정 (아이폰 단축어 자동화)

앱 자체 푸시 대신 아이폰 기본 자동화를 사용합니다.

1. **단축어** 앱 → **자동화** 탭 → **+**
2. **특정 시간** 선택 (예: 오전 8시, 매일) → "즉시 실행" 켜기
3. 동작 추가: **"알림 표시"** → 내용 예: "아침 약 드실 시간이에요 💊"
4. 저녁 등 다른 복용 시간도 같은 방식으로 추가

복용 시간에 알림을 받으면 → 홈 화면의 **복약 체크** 앱을 열어 체크하면 됩니다.
(iOS 17 이상에서는 자동화 동작에 **"앱 열기"**로 홈 화면 웹앱을 직접 열도록 추가할 수도 있습니다.)

## 파일 수정 후 배포 시 주의

`sw.js`의 `CACHE_VERSION` 값을 올려야(예: `v1` → `v2`) 이미 설치한 기기에 새 버전이 반영됩니다.
